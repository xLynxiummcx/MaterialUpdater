# MaterialUpdater For Termux

### a simple script that updates 1.20.80 material bin files to 1.20.20+

# How to Use

> git clone
> 1.run ``setup.sh``

> 2.put your files[materialbin]in unpack

> 3.then run ``upd.sh``

> 4. your updated file will be in ``Repacked`` folder
