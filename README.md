# MaterialUpdater For Termux

### a simple script that updates 1.20.80 material bin files to 1.21.20+

### How is works
1.unpacks old files
2.repack to latest\(ϋ)/♩
# How to Use

> git clone
> 1.run ``setup.sh``

> 2.put your files[materialbin]in unpack

> 3.then run ``upd.sh``

> 4. your updated file will be in ``Repacked`` folder
